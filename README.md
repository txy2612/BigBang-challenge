# BIG BANG Challenge

A JavaScript program that generates an array of values from 1 to 100.

## Rules

- Numbers divisible by 3 are replaced with `"BIG"`.
- Numbers divisible by 5 are replaced with `"BANG"`.
- Numbers divisible by both 3 and 5 are replaced with `"BIGBANG"`.
- Other numbers remain as number strings.

The generated result is saved in `output.json`.

## Requirements

- Node.js

## How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/txy2612/BigBang-challenge.git
   ```

2. Enter the project folder:

   ```bash
   cd BigBang-challenge
   ```

3. Run the script:

   ```bash
   node index.js
   ```

4. Open `output.json` to view the generated result.
